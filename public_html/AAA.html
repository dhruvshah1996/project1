AAA Testing (ARRANGE-ACT-ASSERT)

What is Testing ?

A test is a procedure that exercises a behavior to determine if the behavior functions correctly. There are several different kinds of tests, like unit tests, integration tests, or end-to-end tests, but all functional tests do the same basic thing: they try something and report PASS or FAIL. On this page we will learn about AAA Testing which is a type of unit test written to make sure your programs basic functions are doing what it is supposed to do.
